# Nintendo NES (NESTang)

**Status: In Progress**

The NES core targets the Nintendo Entertainment System (1983) — featuring the 2A03 CPU, PPU, APU, and broad mapper support.

Based on [NESTang](https://github.com/nand2mario/nestang) by nand2mario — being ported to the Retrocade.

---

:::note Coming Soon
The NES core is currently in active development. This page will be updated when the core is available for the Retrocade.

Follow progress at [GitHub](https://github.com/Papilio-Retrocade) or join the discussion at [community.papilioworks.com](https://community.papilioworks.com).
