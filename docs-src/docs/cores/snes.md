# Super Nintendo (SNESTang)

**Status: In Progress**

The SNES core targets full compatibility with the Super Nintendo Entertainment System (1990) — including Mode 7, SuperFX, and all official mappers.

Based on [SNESTang](https://github.com/nand2mario/snestang) by nand2mario — being ported to the Retrocade.

---

:::note Coming Soon
The SNES core is currently in active development. This page will be updated when the core is available for the Retrocade.

Follow progress at [GitHub](https://github.com/Papilio-Retrocade) or join the discussion at [community.papilioworks.com](https://community.papilioworks.com).
:::
