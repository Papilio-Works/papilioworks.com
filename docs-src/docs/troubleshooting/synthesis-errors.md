## Common Error Categories

- **Resource overuse** — design exceeds available LUTs or DSPs
- **Timing violations** — critical path too slow for target clock
- **Undefined signals** — undriven nets or missing connections
- **Placement failures** — conflicting I/O constraints in the CST file
