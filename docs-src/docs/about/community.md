# Community

---

## Forum

The primary Papilio Works community lives at:

**[community.papilioworks.com](https://community.papilioworks.com)**

Categories:
- **Support & Help** — setup questions, troubleshooting (AI bot operates here)
- **Show & Tell** — share your builds, photos, and projects
- **Core Development** — for core porters and FPGA developers
- **Announcements** — official updates from Papilio Works
- **Feature Requests** — structured feedback and ideas

---

## GitHub

All code is open source and lives across two GitHub organizations:

- **[Papilio-Retrocade](https://github.com/Papilio-Retrocade)** — product repos: core ports, FPGA-Companion, hardware
- **[Papilio-Labs](https://github.com/Papilio-Labs)** — platform libraries: Wishbone bus, tooling

Issues, PRs, and discussions welcome.
