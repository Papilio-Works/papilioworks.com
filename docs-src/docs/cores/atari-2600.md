# Atari 2600 (A2600Nano)

**Status: ✓ Working**

The A2600 core runs the Atari 2600 (1977) — a full cycle-accurate implementation of the TIA (Television Interface Adaptor), RIOT (RAM I/O Timer), and 6507 CPU.

Based on [A2600Nano](https://github.com/Papilio-Retrocade/A2600Nano) — ported to the Retrocade by Papilio Works.
